# Iter 083 Executive Report

This round tested whether the cloud event branch is mainly a `QQQ/XLK` market-state trade. The answer is no.

`platform5 pre1 hold3` remained the winner. On the recent broad sample, the control returned `100.059%` with `10.1%` drawdown. Requiring both `QQQ` and `XLK` to be positive reduced that to `78.127%`; forcing both to be weak collapsed it to `5.346%`. On the longer `2024-2025` aggregate, the same pattern held: control `93.491%`, positive tape `84.154%`, weak tape `-1.873%`.

This means the event edge is real but not explained by simple broad tech-tape gating. Combined with the prior round, we can now say the branch is neither a simple `pre-earnings pullback rebound` nor a simple `strong tape` continuation trade. The next useful work is richer event metadata, not more coarse state filters.
